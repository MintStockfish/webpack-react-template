# Шаблон Вебпак

Этот проект представляет собой шаблон для быстрого начала работы с [Webpack](https://webpack.js.org/). Он включает в себя базовую конфигурацию для сборки TypeScript + React, и других ресурсов, а также инструменты для тестирования, линтинга и форматирования кода, а также **весьма** чистый корень проекта.

## Введение

Шаблон Вебпак предоставляет готовую конфигурацию для сборки фронтенд-проектов. Он поддерживает:

- **TypeScript** и **JavaScript**
- **React**
- **Стилизацию** с помощью CSS
- **Оптимизацию** для продакшн-сборки
- **Тестирование** с использованием Jest
- **Линтинг** и **форматирование** кода
- **Husky**

## Таблица скриптов

---

| Команда              | Описание                                                                  |
| :------------------- | :------------------------------------------------------------------------ |
| `npm start`          | Запуск сервера разработки (**Webpack Dev Server**) с поддержкой **HMR**.  |
| `npm run build`      | Сборка проекта в режиме **development** (без минификации).                |
| `npm run build:prod` | Финальная **production** сборка (максимальная оптимизация и минификация). |
| `npm run analyze`    | Сборка в режиме продакшена с запуском **Bundle Analyzer**.                |
| `npm run test`       | Запуск всех тестов через **Jest** в виртуальной среде **JSDOM**.          |
| `npm run lint`       | Статический анализ кода с помощью **ESLint**.                             |
| `npm run lint:fix`   | Поиск и **автоматическое исправление** ошибок линтером.                   |
| `npm run format`     | Принудительное форматирование всего кода через **Prettier**.              |
| `npm run prepare`    | Служебная команда для инициализации **Husky** (Git Hooks).                |

---

Собранные файлы будут находиться в директории `dist`.

## Структура проекта

```

.
├── config/ # Конфигурационные файлы
│ ├── webpack/ # Конфигурация Webpack
│ ├── jest/ # Конфигурация Jest
│ └── ...
├── src/ # Исходный код
│ ├── assets/ # Статические ресурсы
│ ├── public/ # Публичные файлы
│ ├── App.tsx # Корневой компонент
│ ├── index.html # HTML-шаблон
│ ├── index.tsx # Точка входа
│ └── ...
├── package.json # Зависимости и скрипты
├── tsconfig.json # Конфигурация TypeScript
└── README.md # Документация

```

### Описание ключевых файлов

- **`src/index.tsx`**: Точка входа в приложение.
- **`src/App.tsx`**: Корневой компонент React.
- **`src/index.html`**: HTML-шаблон.
- **`config/webpack/webpack.config.ts`**: Конфигурация Webpack.

## Конфигурация

### Webpack

Конфигурация Webpack находится в `config/webpack/webpack.config.ts`. Вы можете изменить её для добавления новых лоадеров или плагинов.
